Final Project Proposal

Application Overview

This final project will be a simple version of the classic game "Space Invaders". There will be a small spaceship at the bottom of the screen, that will be shooting up into a group of UFO's. The UFO's will also be shooting projectiles down at the user. The user will accumalate points as he shoots down UFO's. The game ends when the user gets hit 3 times.

Technology Overview

This game will be programmed in Java or Javascript. It will be hosted on Azure.
